# VULASTOCK - COMECE EM 5 MINUTOS

## ⚡ Roteiro Ultra-Rápido

### 1️⃣ Instalar (2 minutos)
```bash
npm install
```

### 2️⃣ Configurar (1 minuto)
Crie arquivo `.env.local` com:
```env
EXPO_PUBLIC_SUPABASE_URL=https://seu-projeto.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=sua-chave-aqui
EXPO_PUBLIC_APP_ENV=development
```

### 3️⃣ Rodar (2 minutos)
```bash
npm start
```

Escaneie o QR code com **Expo Go** no celular.

---

## ✅ Verificação Rápida

Quando abrir:
- Login com email/senha (ou crie conta)
- Clique em "Nova Entrada"
- Selecione um produto
- Pronto! ✨
